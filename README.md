# student_record_management_system
hands on task
This program is a console-based Student Record Management System.
It uses a list to store multiple student records and dictionaries to store individual student details.
Each student record contains name, roll number, marks, total, average, and grade.
The program uses functions like add_student(), view_students(), search_student(), and class_statistics().
A while loop is used to continuously display the menu until the user selects Exit.
A for loop is used for searching and displaying student records.
Grades are assigned based on the average marks using conditional statements.
Duplicate roll numbers are prevented using a validation check.
AI assistance was used to structure the program logic and improve formatting.
